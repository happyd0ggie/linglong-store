# [简易玲珑商店] linglong-store
 
这里是一个使用electron+vite构建出的简单玲珑客户端程序，质量没保障，非官方制作！
里面包含着大量BUG,欢迎使用App内置的问题反馈通道，作者会认真审视各位的反馈意见
